PASSIVE SONAR SENSITIVITY
==========================
Note: Specific values not defined in current code implementation.
Base sensitivity appears to be class-dependent with towed array multipliers.

ADDITIONAL SYSTEMS
==================
- Towed Array: 1.0x sensitivity multiplier, 0.7x speed penalty when deployed
- SCAV Mode: 2.0x speed multiplier, 10.0x noise penalty, 0.3x maneuverability
- Emergency Repairs: Available on all classes
- Knuckle Creation: Sharp turns at speed >3 knots with turn rate >50

DESIGN NOTES
=============
- All speeds reduced by 25% from original specifications (completed 2025-09-10)
- Tornado is the only mine-laying capable submarine
- Lightning has anti-torpedo light torpedoes for defensive operations
- Heavy torpedoes only available on Typhoon, Hurricane, and Tempest classes
- Command systems unique to Tempest class for fleet coordination
- All classes use the same 3D model geometry scaled by dimensions
