Submarine Control Test

Open the game at http://localhost:8000

Then open browser console (F12) and run these commands:

Test Initial Orientation:

// Check if submarine starts level
if (window.playerSubmarine && window.playerSubmarine().mesh) {
    const mesh = window.playerSubmarine().mesh;
    console.log('Initial rotation:', {
        pitch: (mesh.rotation.x * 180/Math.PI).toFixed(1) + '°',
        yaw: (mesh.rotation.y * 180/Math.PI).toFixed(1) + '°', 
        roll: (mesh.rotation.z * 180/Math.PI).toFixed(1) + '°'
    });
}

Test Roll Controls:

Move your mouse left and right from center, then run:

// Check roll after maneuvering
if (window.playerSubmarine && window.playerSubmarine().mesh) {
    const mesh = window.playerSubmarine().mesh;
    console.log('Current rotation:', {
        pitch: (mesh.rotation.x * 180/Math.PI).toFixed(1) + '°',
        yaw: (mesh.rotation.y * 180/Math.PI).toFixed(1) + '°', 
        roll: (mesh.rotation.z * 180/Math.PI).toFixed(1) + '°'
    });
}
